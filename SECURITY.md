# Security Policy

## Reporting a Vulnerability

If you discover a security issue in Nook, please do not open a public issue first.

Report it privately to the maintainer with:

- affected version
- impact summary
- reproduction steps
- suggested mitigation if available

If a dedicated security contact is added later, update this file with the official address.

## Scope

Please report issues such as:

- unsafe handling of credentials
- command injection
- insecure file permissions
- unintended disclosure of config or history data
